# 💣 经典扫雷游戏 💎

一个界面精美、功能丰富的Web版扫雷游戏，具有现代化设计和完整的游戏体验。

## 🎮 游戏特色

### 核心玩法
- **经典扫雷**: 传统的扫雷游戏规则
- **智能安全区**: 首次点击确保不会踩到地雷
- **自动展开**: 点击空白区域自动展开相邻格子
- **旗帜标记**: 右键标记可疑地雷位置
- **实时计时**: 精确的游戏计时系统

### 四种难度等级

#### 🟢 简单模式
- **网格**: 9×9
- **地雷**: 10个
- **特点**: 适合新手入门

#### 🟡 中等模式
- **网格**: 16×16
- **地雷**: 40个
- **特点**: 平衡的挑战性

#### 🔴 困难模式
- **网格**: 16×30
- **地雷**: 99个
- **特点**: 考验策略思维

#### 🟣 专家模式
- **网格**: 20×24
- **地雷**: 130个
- **特点**: 终极挑战

## 🎯 游戏机制

### 基本操作
- **左键点击**: 揭开格子
- **右键点击**: 标记/取消标记旗帜
- **数字显示**: 显示相邻地雷数量
- **自动展开**: 空白格子自动展开周围区域

### 获胜条件
- 揭开所有非地雷格子
- 所有地雷被正确标记（可选）

### 失败条件
- 点击到地雷格子

## 🏆 成就系统

- **初次胜利**: 完成第一局游戏
- **闪电之手**: 30秒内完成简单难度
- **完美主义者**: 不使用旗帜完成游戏
- **幸运七**: 连胜7局
- **专家大师**: 完成专家难度

## 📊 统计功能

### 详细数据追踪
- **游戏次数**: 总游戏局数
- **获胜次数**: 成功完成的局数
- **胜率**: 获胜率百分比
- **最佳时间**: 各难度最快完成时间
- **连胜记录**: 当前连胜和最高连胜

### 数据持久化
- 本地存储统计数据
- 跨会话保持记录
- 成就进度保存

## 🎲 游戏策略

### 基础技巧
1. **从角落开始**: 角落和边缘通常更安全
2. **数字分析**: 利用数字推断地雷位置
3. **标记策略**: 及时标记确定的地雷
4. **模式识别**: 学习常见的地雷分布模式

### 进阶策略
1. **概率计算**: 基于已知信息计算概率
2. **逻辑推理**: 使用排除法确定安全区域
3. **风险评估**: 在必要时承担计算过的风险
4. **时间管理**: 平衡速度与准确性

## 🎮 如何开始

1. 打开 `index.html` 文件
2. 选择合适的难度等级
3. 点击任意格子开始游戏
4. 使用逻辑推理找出所有地雷
5. 享受经典扫雷的乐趣！
